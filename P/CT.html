Control Tower
Project to improve the service and minimizing the time of delivery in an Manufacturing company. Developed a web tool which interface with Oracle ERP with several modules (suppliers, buyers, shipments, forecasts, on transit, etc.) Allowing monitoring in single traffic lights status, red light for expired due dates or lack of materials, green for forecasted fully deliveries on time. System has a complex security in place connecting to Oracle and using users responsibilities and modules to reflect same permission on Control Tower. Special setups apply to adecuate screening to each roles and specific needs.
